# hao-backprop-test

test project for backprop integration. Do not touch!

## Description

Hello World HTTP server implemented in Python using Flask. Responds to all requests with "Hello, World!" on port 3000.

## Prerequisites

- Python 3.12+

## Setup

```bash
pip install -r requirements.txt
```

## Run

```bash
python app.py
```

Server will start at http://127.0.0.1:3000/
